# Library Management System

A simple and efficient **Library Management System** built using **PHP** for the backend and **MySQL** for the database. This project helps libraries manage books, users, and transactions in an organized way, reducing manual effort and improving tracking of borrowed and returned books.

## Features

- Add, update, and delete books
- Manage library users (members)
- Track borrowing and returning of books
- Search for books and users
- Dashboard with statistics (total books, users, borrowed books)

## Technology Stack

- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git & GitHub
